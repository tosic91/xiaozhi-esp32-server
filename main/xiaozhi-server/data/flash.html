Flash Mochi vào Xiaozhi

Cắm ESP-BOX-3 qua USB-C → bấm nút bên dưới → chọn cổng → chờ flash xong!

⚠️ Trình duyệt không hỗ trợ. Vui lòng dùng Chrome hoặc Edge trên máy tính.

🔒 Trang cần HTTPS để sử dụng Web Serial.

📋 Hướng dẫn

1

Cắm USB-C vào Xiaozhi

Dùng dây USB-C DATA (không phải sạc). Cắm vào cổng USB trên ESP-BOX-3.

2

Vào Boot Mode (nếu cần)

Giữ nút BOOT → nhấn RST → thả BOOT. Thiết bị sẽ vào chế độ flash.

3

Bấm "Flash Firmware" ở trên

Chọn cổng serial (thường là "USB JTAG" hoặc "CP210x") → chờ flash hoàn tất (~2 phút).

4

Cấu hình WiFi

Sau khi flash xong, kết nối WiFi "Xiaozhi-XXXX" → mở 192.168.4.1 → nhập WiFi nhà.
